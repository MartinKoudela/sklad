# DeckShop Storage

Konzolový systém v C pro správu skladu skateboardových desek.

## Funkce

- **Zobrazení skladu** — výpis všech desek s informacemi a dostupnosti
- **Vyhledávání** — hledání podle názvu, značky, materiálu nebo ceny
- **Detailní vyhledávání** — hledání s výpisem všech parametrů desky
- **Přidání desky** — přidání nové desky do skladu
- **Smazání desky** — vyhledání a odstranění desky ze skladu
- **Úprava desky** — změna názvu, značky, ceny, velikosti nebo materiálu


## Ovládání

Po spuštění se zobrazí menu s možnostmi 0–6. Volba se provádí zadáním čísla a
potvrzením Enterem.
